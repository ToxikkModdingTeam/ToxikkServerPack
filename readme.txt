Run setup.cmd to install everything needed for a TOXIKK dedicated server.

Tools included in this installation package:
- steamcmd (command line client for Steam): https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip
- Toxikk Server Launcher 2.12: https://github.com/ToxikkModdingTeam/ToxikkServerLauncher
- nginx 1.10.0 (HTTP server): http://nginx.org/
- curl 7.48.0 (file download tool): https://curl.haxx.se/
- fart 1.99b (find and replace text): http://fart-it.sourceforge.net/
