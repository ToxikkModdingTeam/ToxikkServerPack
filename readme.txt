Download the latest installation package from https://github.com/ToxikkModdingTeam/ToxikkServerPack/releases
and run setup.cmd to install everything needed for a TOXIKK dedicated server with custom content support.

Tools included in this installation package:
- steamcmd (Steam command line client): https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip
- nginx 1.11.3 (HTTP server): https://nginx.org/en/download.html
- winws 1.19-SNAPSHOT build for .NET 4.0 (Windows Service wrapper for nginx): https://github.com/kohsuke/winsw
- curl 7.50.1 (file download tool): https://curl.haxx.se/download.html
- fart 1.99b (find and replace text): http://fart-it.sourceforge.net/
- Toxikk Server Launcher 2.21: https://github.com/ToxikkModdingTeam/ToxikkServerLauncher
